## TABLAS

```


|MARKDOWN	 |RESULTADO    |
|:--------- :|:---------:  |
|\*cursiva*  |	*cursiva*  |
|\_cursiva_  |	_cursiva_  |
|\**negrita**|	**negrita**|
|\__negrita__|	__negrita__|
|\~tachado~  |  ~~TACHADO~~|

```

|    MARKDOWN     |  RESULTADO  |
| :-------------: | :---------: |
|   \*cursiva\*   |  _cursiva_  |
|   \_cursiva\_   |  _cursiva_  |
| \*\*negrita\*\* | **negrita** |
| \_\_negrita\_\_ | **negrita** |
|   \~tachado~    | ~~TACHADO~~ |

## Listas desordenadas

Para crear listas desordenadas utiliza \* asteriscos, - guiones, o + símbolo de suma.

- Elemento de lista 1
- Elemento de lista 2

* Elemento de lista 3
* Elemento de lista 4

- Elemento de lista 5
- Elemento de lista 6

---

## Listas ordenadas

Para crear listas ordenadas debes utilizar la sintaxis de tipo: «número.» 1.

1. Elemento de lista 1
2. Elemento de lista 2
   - Elemento de lista 3
   - Elemento de lista 4
     1. Elemento de lista 5
     2. Elemento de lista 6
