> *Aqui tienes el codigo basico para crear una pagina web con ***html.***


```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>My Web</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
```
