 ## CREAR PAGINA

    CREAR UNA CARPETA DEL PROYECTO:

ABRIR LA CONSOLA DE COMANDOS CMD

>En la consola situada en la carpeta del proyecto escribimos el comando, para crear la Pagina.
~~~
>npx create-docusaurus@latest myWeb classic
~~~
